# Keep docs in sync

When changing these files, check if README.md or CLAUDE.md need updating:

- **New/changed endpoint** → update CLAUDE.md endpoint table
- **New/changed CLI command** → update README.md CLI section + CLAUDE.md file list
- **New/changed .env variable** → update README.md Configuration section + .env.example
- **New file added to project** → update CLAUDE.md file list
- **Test count changes significantly** → update CLAUDE.md test count
